# coffee-memo-frontend
このアプリはコーヒーに関するメモを登録できるWebアプリです。  
デモサイト: https://coffee-note-m1kan1.netlify.app

## 使用しているライブラリやフレームワーク
* Nuxt.js
* Vuex
* Axios
* Clarity
* Vuetify

## インストール方法
```
# リポジトリをクローン
$ git clone https://github.com/m1kan1/coffee-note-frontend.git

# プロジェクトのディレクトリに移動
$ cd coffee-note-frontend

# 依存関係をインストール
$ yarn install

# 環境変数を設定
$ cp env.template .env
$ vi .env

# 開発サーバーを起動
$ yarn dev
```
